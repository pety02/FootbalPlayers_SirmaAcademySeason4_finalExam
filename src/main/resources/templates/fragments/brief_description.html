Application brief description:

Implemented By

Petya Licheva

This is a Spring Boot Application that shows matches, teams and footballers that are played on the Football Championship 2024 and summarize their results. Also, the application provides options for adding, deleting and updating any of the Footballer, Team or Match records. Till now there is no Admin Panel but it is good option because the simple user of the application is not good idea to has the opportunity of database's tables updates. The used database is PostgresSQL.
